# Device Naming Convention

Devices are named after **raccoons and other "forbidden cats"** — nocturnal creatures that look like cats but aren't.

## Theme

- **Raccoons / trash pandas** — primary theme (Procyonidae family)
- **Possums / opossums** — nocturnal scavengers, play dead
- **Red pandas** — the original "firefox"
- **Foxes** — cunning nocturnal scavengers
- **Skunks** — nocturnal, bold, misunderstood
- **Other forbidden cats** — tanuki, coati, kinkajou, binturong, civet, genet, etc.

## Naming Patterns

| Pattern | Example | Notes |
|---------|---------|-------|
| `adjective-behavior` | `sleepy-scavenger` | Alliterative preferred |
| `material-noun` | `titanium-trashcan` | References device material |
| `scientific-name` | `procyonid-trailblazer` | Procyonid = raccoon family |
| `animal-location` | `couch-possum` | Animal + where it lives |
| `wordplay` | `paws-on-deck` | Puns welcome |
